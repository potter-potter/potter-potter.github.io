Income By Zip Api Sample Data

JSON Returned From Api

Data returned from the Api will look like the below.

							Zipcode

							Income150kTo200k
							Number of households with yearly income $150,000 to $200,000
							

							

							IncomeAbove200k
							Number of households with yearly income over $200,000
							

							

							Population
							Number of people in zipcode
							

							

							Asian
							Population that is asian
							

							

							Black

PopulationDensity
