Income By Zip Api FAQ

Where does the data come from?

All data comes from the United States Census Bureau and is the latest available (currently 2018).

What is the API endpoint?

The endpoints and a swagger (OpenApi) doc are provided upon purchase. Don't worry, its a very simple api.

What format does the data come in?

It comes in json format, which is the industry standard. See the sample data page here.

What is a ZCTA vs a Zip code?

Data from the US Census comes in the form of ZCTA codes. Thats just the way they do it. In most cases they are completely compatible with zip codes. The exception would be Post Office boxes which have no demographic data and therefore have no ZCTA data.

How can I get and inovice or receipt?

Check your purchase email, it should have links to create an invoice and a receipt.

How do I cancel?

You can cancel anytime. Check your purchase email, it should have links to create an invoice and a receipt. Your api key will be good till the end of the month that you cancel.

How do I use the api?

You will receive an API key and endpoint by email when you purchase. Smashing Magazine has a good explanation on how to use REST Apis here. You can call up to 10 zipcodes in one Api call. It will looks something like this: 

https://api.EXAMPLE.com/zipcode?zip=93117&zip=2817&zip=2819&zip=2822&api_key=111-111-111

How many API calls do I get per month?

That depends on what you purchased. Please see info on the pricing page. If you need more please contact us.
